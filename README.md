## ğŸ® How to Play

### Objective
Be the first player to get three of your marks (X or O) in a row - horizontally, vertically, or diagonally.

### Controls

**Desktop & Mobile:**
- **Click/Tap Empty Square** - Place your mark (X or O)
- **Restart Button** - Start a new game
- **Mode Toggle** - Switch between AI and 2-Player mode (if available)

### Game Rules
- Players alternate turns placing their mark on empty squares
- **X always goes first**
- First player to align three marks in a row wins
- If all 9 squares are filled with no winner, the game is a **draw**
- Winner is highlighted with animation

### Winning Patterns
There are 8 possible ways to win:
- **3 Horizontal Rows** (top, middle, bottom)
- **3 Vertical Columns** (left, center, right)
- **2 Diagonals** (top-left to bottom-right, top-right to bottom-left)
